# Trocador de Calor
### PI 1 - Turma A - 2018/2

## Instalação

É necessário que você tenha o Node e o NPM instalados na sua máquina

Clone o repositório

```
git clone https://github.com/szwacz/electron-boilerplate.git
```

Em seguida instale as dependências
```
npm install
```

Pode iniciar a aplicação
```
npm start
```
